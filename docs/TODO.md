## 3. Сила босса показывает 63 вместо 154
- **Что**: Босс Двуязыкий имеет буквы [А,Е,Б,В,Г], сила должна быть 33+28+32+31+30=154. Но авто-бой показывает 63 (как будто только 2 буквы).
- **Причина**: Возможо спавнер леса передаёт не те буквы, или `_letters` не загружается из `monsters.json` правильно для босса.
- **Файлы**: `monster_spawner.gd` (forest setup), `monster_base.gd:_ready()` (загрузка letters)
- **Проверить**: `window.gameMonsterStates` — какие буквы у босса в runtime

## 4. Рекрутовый монстр атакует ближайшего `!`, а не босса
- **Что**: Завербованный `?` монстр в лесу нашёл ближайшего `!` (Тревога, 63 силы) вместо босса Двуязыкого (154 силы). Бот показал "Армия 63 vs 63" вместо "Армия X vs 154".
- **Причина**: `_find_nearest_hostile(350.0)` находит ближайшего в радиусе 350px. Босс может быть дальше.
- **Решение**: Увеличить радиус поиска для боссов, или приоритизировать боссов.
