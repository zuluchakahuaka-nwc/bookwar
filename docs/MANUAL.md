# BOOKWAR — Руководство игрока

> Последнее обновление: 2026-06-15
> Версия: 0.3 (Бой + таймер, буквы вверху, квесты, прогрессия карт)

---

## 1. ЧТО ЭТО ЗА ИГРА

В мире запретили буквы. Их разбросали по тёмным землям. Ты — герой, который отправляется вернуть все 33 буквы русского алфавита и восстановить истину.

Буквы — это одновременно оружие, защита и валюта. Без букв ты никто. Твой путь начинается с нуля: ни одной буквы, ни одного оружия.

**Жанр**: RPG с карточными боями на основе русского алфавита.
**Платформа**: Браузер (itch.io), ПК, мобильные устройства.

---

## 2. УПРАВЛЕНИЕ

### 2.1 Клавиатура (ПК)

| Клавиша | Действие |
|---------|----------|
| **W** | Движение вверх |
| **A** | Движение влево |
| **S** | Движение вниз |
| **D** | Движение вправо |
| **E** | Взять/взаимодействовать с объектом |
| **I** | Открыть/закрыть инвентарь (сумку) |
| **T** | Начать разговор (если рядом монстр/NPC и есть многоточие) |
| **Пробел** | Пауза (в мире) / Подтвердить ход в бою / продолжить диалог |

### 2.2 Мышь / сенсор (мобильные)

На экране всегда видны экранные кнопки:

- **D-pad** (нижний левый угол) — стрелки движения (W/A/S/D)
- **Кнопки действий** (нижний правый угол):
  - «Взять [E]» — подобрать предмет
  - «Сумка [I]» — открыть инвентарь
  - «Речь [T]» — начать диалог

Кнопки работают так же, как клавиатура. Можно играть полностью мышью/пальцем.

### 2.3 Подсказка

В верхней части экрана всегда видна строка:
```
WASD / кнопки — движение   |   E — взять   |   I — инвентарь   |   T — речь
```

## 3. РЕСУРСЫ

### 3.1 Точки (`.`)

Точка — базовый ресурс, который лежит на земле в Светлой Долине.

- Подходишь к точке → нажимаешь **E** (или подходишь вплотную — авто-сбор).
- В HUD (верхний левый угол) видно счётчик: **`.: 5`** — у тебя 5 точек.

### 3.2 Многоточие (`...`)

Многоточие — ключевой ресурс старта игры.

- **3 точки автоматически комбинируются в 1 многоточие.**
- Счётчик в HUD: **`...: 2`** — у тебя 2 многоточия.
- Многоточие тратится на **разговор** с монстрами и NPC (клавиша **T**).
- **Пока у тебя нет букв — многоточие единственный способ взаимодействовать с врагами.** Убедить, избежать боя, получить подсказку.

### 3.3 Буквы

Буквы — это оружие, защита и валюта. Всего 33 буквы русского алфавита.

- На старте у тебя **0 букв**.
- Буквы находятся в тайниках, выпадают из монстров после победы, даются NPC.
- **Уровень буквы = количество собранных копий.**
  - 1 копия А → уровень 1 → урон 33 × 1 = 33
  - 3 копии А → уровень 3 → урон 33 × 3 = 99
- При атаке/защите буква **не теряется**. Уровень — перманентный.

### 3.4 Знаки препинания

Особые карты, не привязанные к алфавиту:

| Знак | Эффект |
|------|--------|
| `.` | Точка — базовый ресурс |
| `...` | Многоточие — разговор с монстрами/NPC |
| `,` | Запятая — замедляет врага на 1 ход (скорость −5) |
| `-` | Дефис — уклонение от одной атаки |
| `:` | Двоеточие — раскрывает намерения врага на след. ход |
| `;` | Точка с запятой — позволяет сыграть 2 буквы за 1 ход |
| `""` | Кавычки — отражает следующую атаку обратно |
| `—` | Тире — пробивает защиту, игнорирует щит |
| `()` | Скобки — ловушка, наносит урон при атаке на тебя |

## 4. БУКВЫ — ОРУЖИЕ И ЗАЩИТА

### 4.1 Типы букв

| Тип | Буквы | Роль |
|------|-------|------|
| **Гласные** | А, Е, Ё, И, О, У, Ы, Э, Ю, Я | **Атака** — наносят урон |
| **Согласные** | Б, В, Г, Д, Ж, З, Й, К, Л, М, Н, П, Р, С, Т, Ф, Х, Ц, Ч, Ш, Щ | **Защита** — создают щит |
| **Особые** | Ъ, Ь | **Баффы** — усиливают след. ход |

### 4.2 Сила и скорость

Позиция буквы в алфавите определяет её характеристики:

- **А** (позиция 1): сила = 33, скорость = 1 (самая медленная, но мощнейшая)
- **Я** (позиция 33): сила = 1, скорость = 33 (молниеносная, но слабая)

**Принцип баланса**: чем сильнее буква, тем медленнее она ходит в бою.

### 4.3 Цвета карточек в бою

- **Красно-оранжевые** — гласные (атака)
- **Сине-стальные** — согласные (защита)
- **Фиолетовые** — особые (Ъ, Ь — баффы)

### 4.4 Баффы Ъ и Ь

- **Ъ** (твёрдый знак) — усиливает следующую согласную защиту (×1.5)
- **Ь** (мягкий знак) — усиливает следующую гласную атаку (×1.5)

Выкладываешь Ь → следующий ход кладёшь гласную → урон ×1.5.

## 5. БОЙ

### 5.1 Как начинается бой

Агрессивные монстры (например `!`) атакуют при контакте. Ты переходишь на экран боя.

### 5.2 Экран боя

```
┌────────────────────────────────────┐
│  Враг: TestBug    HP: 50/50  Щит:0│
│                                    │
│  [ Лог боя ]                       │
│  Бой начался: TestBug              │
│  А нанёс 66 урона → enemy          │
│                                    │
│  HP: 99/100  Щит: 0                │
│                                    │
│  [Б Lv1] [А Lv2] [✦БАМ 170]       │
│                                    │
│  [Подтвердить ход]  [Бежать]       │
└────────────────────────────────────┘
```

- **Вверху**: имя врага, его HP и щит.
- **Центр**: лог боя (кто что сделал, сколько урона).
- **Внизу слева**: твои карты — буквы (с уровнем) и заклинания (с силой).
- **Внизу справа**: кнопки «Подтвердить ход» и «Бежать».

### 5.3 Как ходить

1. **Выбери карту** — кликни на букву или заклинание (или нажми соответствующую кнопку).
2. **Подтверди ход** — нажми «Подтвердить ход» или клавишу **E** / **Пробел**.
3. Ход разрешается по скорости: быстрые буквы бьют первыми, медленные — последними.
4. После разрешения хода виден результат: HP изменился, в логе — кто нанёс урон/щит.

### 5.4 Победа и поражение

- **Победа**: HP врага ≤ 0 → выпадает лут (1-3 буквы из набора врага).
- **Поражение**: твой HP ≤ 0 → бой проигран.
- **Бегство**: кнопка «Бежать» — покинуть бой без результата.

### 5.5 Стратегия

- Быстрые буквы (Э, Ю, Я) бьют первыми — хорошо для добивания.
- Мощные буквы (А, Б) бьют последними, но сильно — риск получить урон до хода.
- Оптимально: баланс скорости и силы.
- **Ъ + согласная** = усиленный щит.
- **Ь + гласная** = усиленный удар.

## 6. ЗАКЛИНАНИЯ

Заклинания — это **слова** из 2-4 букв. Сильнее, чем отдельные буквы.

### 6.1 Как работают

- Нужно **открыть** (купить) рецепт заклинания за валюту (буквы).
- Для каста нужны все буквы слова (уровень ≥ 1 у каждой).
- Заклинание **стоит 1 ход**, но эффект комбинированный.
- Буквы при касте **не расходуются**.

### 6.2 Формула силы

```
сила = (сумма base_power букв) × множитель_заклинания × средний_уровень_букв
```

### 6.3 Реестр заклинаний

| Слово | Буквы | Эффект | Множитель |
|-------|-------|--------|-----------|
| **БАМ** | Б+А+М | Двойной урон | ×2.0 |
| **БУМ** | Б+У+М | Игнорирует щит врага | ×1.8 |
| **ЩИТ** | Щ+И+Т | Двойной щит | ×2.5 |
| **БОГ** | Б+О+Г | Лечение +40 HP | ×1.0 |
| **ДЫМ** | Д+Ы+М | Уклон от след. атаки | ×1.0 |
| **ЖАР** | Ж+А+Р | Поджог — урон 3 хода | ×1.5 |
| **ЛУК** | Л+У+К | Дальний выстрел, враг не отвечает | ×1.6 |
| **РОГ** | Р+О+Г | Бафф ×2 на след. ход | ×2.0 |
| **ХРЕН** | Х+Р+Е+Н | Тройной удар | ×2.2 |
| **ФИГ** | Ф+И+Г | Отражение след. атаки | ×1.0 |

### 6.4 Валюта

Каждая буква имеет ценность = позиция в алфавите:
- А = 1 (самая дешёвая), Я = 33 (самая дорогая).
- Кошелёк = сумма (позиция × уровень) по всем буквам.

Пример: А(ур.3) + О(ур.2) + Я(ур.1) = 1×3 + 16×2 + 33×1 = **68 монет**.

## 7. ДИАЛОГИ

### 7.1 Как начать разговор

1. Собери **3 точки** → получится **1 многоточие**.
2. Подойди к монстру или NPC.
3. Нажми **T** (или кнопку «Речь»).

### 7.2 Что даёт диалог

- **Избежать боя** — убедить монстра уйти.
- **Получить подсказку** — где найти буквы, куда идти.
- **Получить букву** — монстр может отдать букву после разговора.
- **Завербовать монстра** — см. раздел 12 (Лояльность).
- **Торговать** — обменять точки/буквы.

### 7.3 Закрытие диалога

Диалог закрывается если:
- Нажал **E** или **T** (закрыть).
- **Отошёл** от монстра (вышел из зоны обнаружения) — диалог обрывается.
- **Начал двигаться** (WASD во время диалога = уйти).

При закрытии диалога срабатывает **бросок рекрутинга 50/50** (см. §12.2).

### 7.4 Важно

- **Во время диалога монстры НЕ атакуют.** Диалог ставит бой на паузу.
- Многоточие **тратится** при начале диалога.

## 8. ИНВЕНТАРЬ (СУМКА)

Нажми **I** чтобы открыть/закрыть инвентарь.

В инвентаре видно:
- Собранные буквы (с уровнем)
- Количество точек
- Количество многоточий
- Знаки препинания

## 9. МИР

### 9.1 Светлая Долина (старт)

- Уютная деревня с NPC.
- Луга, ручей, лесок.
- Монстры: `?` (подозревает) и `!` (заметил) — простые, слабые.
- Ресурсы: точки `.` разбросаны по земле.
- Буквы спрятаны — награда за исследование.
- Нет оружия на старте — только сбор и диалог.

### 9.2 Лояльность монстров (Allegiance)

Каждый монстр имеет одно из трёх состояний. **Цвет монстра = его лояльность:**

| Цвет | Состояние | Поведение |
|------|-----------|----------|
| 🔴 **Красный** | ВРАГ (Hostile) | Преследует и атакует игрока |
| 🟢 **Зелёный** | СОЮЗНИК (Recruited) | Следует за героем, сражается за него |
| ⚪ **Серый** | НЕЙТРАЛ (Neutral) | Игнорирует игрока, патрулирует |

- `!` монстры стартуют **красными** (враги).
- `?` монстры стартуют **серыми** (нейтралы).
- Лояльность меняется через **диалог и рекрутинг** (см. §12).

### 9.3 Монстры

**`?` (Вопросительный)** — Серый (нейтрал):
- Патрулирует территорию.
- Игнорирует игрока, если не спровоцировать.
- Можно завербовать через диалог (50% шанс).
- При провале рекрутинга — остаётся нейтралом.

**`!` (Восклицательный)** — Красный (враг):
- Агрессивный, атакует при контакте.
- Быстрый, но слабый.
- Можно завербовать через диалог (50% шанс).
- При провале рекрутинга — остаётся врагом, возобновляет погоню.

### 9.4 Опасные зоны (Danger Zones)

Чем дальше от точки старта — тем больше и опаснее монстры:

| Зона | Дистанция от старта | Монстры |
|------|---------------------|---------|
| Кольцо 1 | 300–600px | `?` (нейтральные, диалог) |
| Кольцо 2 | 600–900px | Микс `?` и `!` |
| Кольцо 3 | 900–1300px | Отряды `!` (агрессивные) |
| Кольцо 4 | 1300+px | Орды `!` |

**Совет**: не уходи далеко от старта без приличного набора букв.

### 9.5 Прогрессия мира

По мере удаления от Светлой Долины мир темнеет:
- Тёмный Лес, Гнилые Болота, Катакомбы, Цитадель...
- Монстры начинают использовать буквы.
- Появляются боссы регионов.
- Финал — Хранитель Запрета (владеет всем алфавитом).

## 10. СОВЕТЫ НОВИЧКУ

1. **Сначала собирай точки.** Без них не будет диалога, а без диалога никак.
2. **3 точки = 1 многоточие.** Накопи хотя бы 2-3 многоточия.
3. **Говори с монстрами.** На старте это единственный способ выжить.
4. **Ищи буквы.** Они спрятаны в Светлой Долине — исследуй каждый угол.
5. **Первые буквы — твоё оружие.** Получив гласную (например А) — можешь атаковать в бою.
6. **Согласные = щит.** Получив согласную — можешь защищаться в бою.
7. **Не дерись без нужды.** Пока букв мало, лучше убеждать через диалог.
8. **Комбинируй Ь + гласная** для усиленного удара.
9. **Открывай заклинания** — они намного сильнее отдельных букв.
10. **Чем глубже в мир — тем опаснее.** Не лезь в Тёмный Лес без приличного набора букв.
11. **Смотри на цвет монстров.** Красный = враг, серый = нейтрал, зелёный = союзник.
12. **Вербуй монстров.** После диалога 50% шанс что монстр присоединится.
13. **Прячься в лесу/воде.** Если тебя преследуют — беги к тёмным тайлам.
14. **Пауза — Пробел.** Можно остановить игру в любой момент (кроме боя).

│                                             │
│              [ Игровой мир ]                │
│                                             │

| Элемент HUD | Значение |

| Кнопки (нижний правый) | Взять / Сумка / Речь |
| `⏸ ПАУЗА` (центр экрана) | Игра на паузе (нажми Пробел чтобы продолжить) |

## 12. ЛОЯЛЬНОСТЬ И РЕКРУТИНГ

### 12.1 Как завербовать монстра

1. Подойди к монстру.
2. Нажми **T** чтобы начать диалог (нужно многоточие `...`).
3. Закрой диалог (**E**, **T**, или отойди).
4. При закрытии — **бросок 50/50**:
   - **Успех (50%)**: монстр становится 🟢 **зелёным** (союзник).
   - **Провал (50%)**: монстр остаётся при текущей лояльности.

### 12.2 Что говорят монстры

- **Успех**: «Возможно ты и не враг... Я с тобой!» → монстр следует за героем.
- **Провал `!`**: «Нет! Ты враг!» → возобновляет погоню.
- **Провал `?`**: «Ладно, иди. Я нейтрален.» → уходит патрулировать.

### 12.3 Банда

Завербованные монстры (🟢 зелёные) **следуют за героем**:
- Бегают рядом с небольшим случайным разбросом.
- Не атакуют игрока.
- Чем больше банда — тем престижнее игрок.
- **В будущих версиях**: банда участвует в бою на стороне игрока.

## 13. СКРЫТИЕ И СТЕЛС

### 13.1 Как спрятаться

Встань на тёмный тайл:
- 🌲 **Лес** (деревья)
- 💧 **Вода**
- 🪨 **Камень** (большие камни)
- 🚧 **Забор**

На траве и тропе — **открыто**, спрятаться нельзя.

### 13.2 Что происходит при скрытии

- Преследующий монстр **теряет тебя из виду**.
- Монстр переходит в режим **поиска** (стоит, оглядывается).
- Через **4 секунды** поиска монстр **теряет интерес** и возвращается к патрулированию.

### 13.3 Стратегия

- Беги к ближайшему лесу или воде когда тебя преследуют.
- Спрячься — монстр отстанет.
- Можно подкрасться к монстрам через тёмные тайлы.

## 14. ПОБЕДА В РЕГИОНЕ

### 14.1 Условия победы (Светлая Долина)

Регион считается зачищённым, когда для **каждого** монстра выполнено одно из:
- **Убит** в бою.
- **Завербован** (присоединился к банде).
- **Нейтрализован** (переболтан до нейтрала).

**Обязательное условие**: игрок должен **подойти** к каждому монстру хотя бы один раз.

### 14.2 Что происходит при победе

На экране: **«ПОБЕДА! Долина зачищена. Союзников: N»**

## 15. ПАУЗА

- **Пробел** — переключает паузу (в игровом мире).
- При паузе: игра замирает, на экране «⏸ ПАУЗА».
- Пауза **не работает** в бою (бой — отдельная сцена).
- HUD продолжает обрабатывать ввод даже в паузе.

## 16. БОЙ — ПОДРОБНО (v0.3)

### 16.1 Как работает бой

1. Монстр `!` касается тебя → начинается бой.
2. Вверху экрана — **таймер 10 секунд**.
3. Вверху — **твои буквы** (с уровнями в скобках).
4. Внизу — кнопки букв (цвет: красный = гласная/атака, синий = согласная/защита).
5. Выбери букву → нажми **«Подтвердить ход»** (или **E**).
6. Если таймер истёк → ход разыгрывается автоматически (враг атакует).

### 16.2 Что делают буквы в бою

| Тип буквы | Что делает |
|-----------|-----------|
| **Гласная** (А, О, Е...) | **Атакует** врага — наносит урон = сила × уровень |
| **Согласная** (Б, К, М...) | **Защищает** — создаёт щит, поглощающий урон |
| **Ъ** (твёрдый знак) | **Бафф защиты** ×1.5 на следующую согласную |
| **Ь** (мягкий знак) | **Бафф атаки** ×1.5 на следующую гласную |

### 16.3 Порядок хода

- Все буквы (твои и врага) выкладываются одновременно.
- Разрешаются **по скорости** (от быстрой к медленной).
- Быстрые буквы (Я=33) ходят первыми, но слабые.
- Медленные буквы (А=1) ходят последними, но мощные.

### 16.4 Победа / Поражение

- **Победа**: HP врага = 0 → лут (1-3 буквы), возврат в мир.
- **Поражение**: HP игрока = 0 → возврат в мир (без лута).
- **Бегство**: нажми «Бежать» → возврат в мир (без лута).

## 17. КВЕСТЫ И ТОРГОВЛЯ (в разработке)

### 17.1 Кто даёт квесты

**Только `?` монстры** — вопросительные знаки. Они нейтральны и готовы говорить.

### 17.2 Типы заданий

| Тип | Описание | Пример |
|-----|----------|--------|
| **Принести** | Собери N нужных букв | «Принеси 3 буквы А» → награда: буква О |
| **Убить** | Очисти карту от N врагов | «Победи 3 монстров !» → награда: заклинание |
| **Купить** | Обменяй точки на букву | «Купи М за 10 точек» |
| **Обменять** | Поменяй одну букву на другую | «Дам К за твою Я» |

### 17.3 Журнал квестов (клавиша Q)

- Показывает активные и выполненные задания.
- Прогресс: «Квестов выполнено: 3/5»

## 18. ПРОГРЕССИЯ — ПУТЬ ПО МИРАМ (в разработке)

Игра — цепочка из **5 карт**. Каждая сложнее предыдущей.

```
Светлая Долина → Лесная Поляна → Дремучие Дубы → Ивовая Гать → Болотная Топь
```

### 18.1 Как перейти на следующую карту

1. Выполни **все квесты** на текущей карте.
2. На северном краю появляется **зелёный коридор**.
3. Войди в коридор → загрузится следующая карта.

### 18.2 Что нового на каждой карте

| Карта | Враги | Особые NPC | Новые буквы |
|-------|-------|-----------|-------------|
| 1. Светлая Долина | ?, ! | — | А, О, М, Б, Я |
| 2. Лесная Поляна | ?, ! + **Длинноязыкий** | Торговец (длинный язык) | + Е, К, Т, Р, Д |
| 3. Дремучие Дубы | + **Слушач** | Огромные уши (дальний слух) | + В, Г, Ж, Л, П, Н |
| 4. Ивовая Гать | + **Зрячий** | Огромные глаза (видит сквозь укрытия) | + З, С, Ф, Х, Ц, Ч, Ш, Щ |
| 5. Болотная Топь | + **Жор** + БОСС | Огромный рот + Хранитель Топи | Все 33 буквы |

### 18.3 Принцип

- Каждая карта: на ~30% больше врагов, квестов и букв.
- К концу 5-й карты ты соберёшь **все 33 буквы**.
- Финальный босс — **Хранитель Топи**, владеющий полным алфавитом.
