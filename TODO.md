## 🔤 КРАФТ БУКВ (в инвентаре)

### C1. Механика трансформации букв
Игрок может изменить одну букву в другую, добавив модификатор:
| Из | + модификатор | → В | шанс успеха |
|----|---------------|-----|-------------|
| Б | + Ь или ' | → П | 70% |
| П | + Ь или ' | → Б | 70% |
| Ф | + Ь | → В | 70% |
| В | + Ь | → Ф | 70% |
| М | + Ь | → Н | 70% |
| Н | + Ь | → М | 70% |
| Щ | + ' | → Ш | 70% |
| Ш | + ' | → Щ | 70% |
| С | + Ь | → З | 60% |
| Д | + Ь | → Т | 60% |

**При неудаче**: буква-донор (модификатор Ь/') ТЕРЯЕТСЯ без получения новой буквы.
**При удаче**: донор расходуется, целевая буква добавляется (или +1 уровень).

### C2. UI крафта
- В инвентаре: вкладка «Крафт»
- Выбрать букву-источник + модификатор → кнопка «Трансформировать»
- Анимация результата (успех/неудача)

### C3. Данные крафта
- `data/crafting.json` — рецепты трансформаций
- Тест через Puppeteer
